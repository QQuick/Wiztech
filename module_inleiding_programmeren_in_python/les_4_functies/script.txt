Intro
=====

Welkom bij les 4 uit de cursus "Inleiding programmeren in Python",
onderdeel van de WizTech lessenserie.
Deze les gaat over functies.
Mijn naam is Jacques de Hooge.

In deze les leer je hoe je programma kunt opsplitsen in overzichtelijke delen.
Dat kan in Python op verschillende manieren, en de manier die in deze les aan de orde komt, zoals gezegd, is het gebruik van functies.

Misschien weet je dat in de wiskunde een functie iets is waarmee je een unieke uitgangswaarde kunt uitrekenenen aan de hand van een of meer ingangswaarden.
Zo kunnen we spreken over de functie Gemiddelde.
Indien we de functie gemiddelde twee waarden meegeven, bijvoorbeeld 3 en 5,
dan is de uitkomst van die functie (3 + 5) / 2 = 8 / 2 = 4.

Een functie is dus eigenlijk een recept om unieke uitgangswaarden te berekenen aan de hand van ingangswaarden.
Je zou ook kunnen zeggen: een functie is een soort fabriekje.
Je stopt de ingangswaarden, de grondstoffen, erin, en dan komen de uitgangswaarden, de eindproducten, eruit.
Laten we eens kijken hoe je zo'n functie kunt maken in Python.
