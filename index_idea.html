Aanbevolen volgorde

De onderwerpen op deze site vormen een samenhangend geheel. Wiskunde staat hier niet op zichzelf maar wordt onmiddelijk toegepast in de techniek. Vanwege deze samenhang is er een soort ideale lijn, een aanbevolen volgorde om door de lessen heen te gaan. Die ideale lijn ziet er als volgt uit:
  • Module 1: Inleiding programmeren in Python - Les 1: Gebruik van Python in een commando venster
  • Module 1: Inleiding programmeren in Python - Les 2: Maken van een programma-bestand en gebruik van if .. elif .. else
  • Module 1: Inleiding programmeren in Python - Les 3: Herhalings-lussen: while en for
  • Module 1: Inleiding programmeren in Python - Les 4: Functies: definitie, aanroep, named en default parameters, *args en **kwargs
  • Module 1: Inleiding programmeren in Python - Les 5: Klassen en objecten: inkapseling, overerving en veelvormigheid
  • Module 1: Inleiding programmeren in Python - Les 6: Modules: (ont)koppeling en cohesie
  • Module 1: Inleiding programmeren in Python - Les 7: Object modellering: van stabiel object model naar flexibele code
  • Module 2: Toegepaste wiskunde - Les 1: Rekenen met reele getallen
  • Module 2: Toegepaste wiskunde - Les 2: Vergelijkingen, matrices en vectoren
